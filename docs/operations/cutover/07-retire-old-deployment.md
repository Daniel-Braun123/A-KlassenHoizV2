# Altdeployment-Ruhestand

Die Hauptdomain zeigt ausschließlich auf V2. Das alte Repository wurde vom Vercel-Projekt getrennt
und kann keine automatischen Deployments mehr auslösen. Frühere Deployment-Artefakte besitzen
keinen Production-Alias; sie bleiben providerseitig nur als unveränderliche Historie/Recovery-
Anker erhalten und werden nicht als aktiver Anwendungspfad verwendet.
