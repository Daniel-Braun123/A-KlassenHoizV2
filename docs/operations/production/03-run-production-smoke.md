# Production-Smoke

Bestanden: direkter Login aller vier Rollen, App-Admin-Zugriff auf globale Verwaltung,
Unsichtbarkeit der privaten Runde für Nichtmitglied und App-Admin, Owner-/Member-Zugriff,
serverseitiger Tippfrist-Deny, Ergebnis-Recalculation für zwei Tipps, deterministische 4-/3-Punkte
und Rangliste. Der Test legte nach der in Aufgabe 2 freigegebenen Fixture keine weiteren Identitäten
oder fachlichen Objekte an.
