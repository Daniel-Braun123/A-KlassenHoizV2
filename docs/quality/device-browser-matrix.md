# Geräte-/Browsermatrix – V1

Automatisiert bestanden: Chromium, Firefox, WebKit, Mobile Chrome und Mobile Safari mit insgesamt
70 E2E-Fällen sowie PWA-Installierbarkeit, Offline-Fallback und Private-Cache-Deny. Gegen die echte
Production-Domain wurden mobile UI, Manifest, Service Worker und Offline-Seite in Chromium geprüft.
Reale Installationen auf iOS-/Android-Hardware wurden mangels physischer Geräte nicht ausgeführt
und sind als befristete V1-Ausnahme gekennzeichnet.
