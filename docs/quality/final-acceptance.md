# Finale V1-Abnahme

Release konsistent. Alle implementierbaren Tasks sind abgeschlossen; Supabase ist migriert und
verifiziert, Production-Smoke bereinigt, Vercel live und GitHub `main` aktuell.

moderierter Fünf-Personen-Test und zwei überschrittene LCP-Laborbudgets. Das Vercel-Git-Autodeploy
für V2 bleibt wegen externer GitHub-App-Berechtigung deaktiviert; der sichere CLI-Deploypfad aus
`main` ist dokumentiert und das alte Repository ist getrennt.
