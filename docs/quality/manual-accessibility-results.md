# Manuelle Accessibility-Abnahme – V1-Entscheidung

Stand: 2026-07-14. Die automatisierten Axe-, Tastatur-, Fokus-, Reflow-, Kontrast- und
Reduced-Motion-Prüfungen sind grün. Eine reale NVDA-/VoiceOver-Abnahme wurde **nicht** durchgeführt.
Der Product Owner hat Screenreader-Funktionalität für diesen privaten V1-Release ausdrücklich
zurückgestellt. Das ist eine Release-Ausnahme, kein WCAG-Nachweis; vor öffentlicher Nutzung ist die
vollständige Matrix aus `manual-accessibility-protocol.md` nachzuholen.
